# SupremeAppsBuilder

## Overview

Build production-grade ChatGPT Apps SDK projects with an adaptive interview, a deterministic refinement loop, modern widget templates, validation, and benchmark artifacts.

## Workflow

1. Collect the spec with `apps_builder_interview`.
2. Confirm and run `apps_builder_generate`.
3. Generate a Node/TypeScript MCP server and React widget scaffold.
4. Validate the output and run the smoke benchmark.
5. Return a deployment URL or local tunnel instructions for ChatGPT Developer Mode.

## Activation

Use when the user wants a new ChatGPT Apps SDK app, wants stronger templates, or needs troubleshooting for scaffold/validation/deployment.
